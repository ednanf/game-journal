## Objective

- This project delivers a full-stack application using the MERN stack, following
  REST API best practices.
- The frontend provides a clean, straightforward interface for interacting with
  the backend, prioritizing simplicity and usability.
- Currently, the auth token is stored in local storage instead of cookies to
  avoid CORS issues, particularly with WebKit on iOS.
